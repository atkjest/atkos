
# 基于qemu的操作系统玩具内核 ![Language](https://img.shields.io/badge/language-c-blue.svg?style=flat-square) ![Language](https://img.shields.io/badge/language-asm-blue.svg?style=flat-square)

* 内存管理:物理内存，虚拟内存的映射以及堆内存的动态分配
* 进程管理:简单的任务调度
* 设备管理:vga字符模式下的显卡驱动
